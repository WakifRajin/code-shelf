# codes folder

Place the code files you want to show on the homepage in this folder.

Then update manifest.json with one object per file:

{
  "name": "example.py",
  "path": "codes/example.py",
  "language": "python",
  "description": "Optional short summary",
  "tags": ["optional", "tags"]
}

After editing manifest.json, click "Refresh Library" in the app.
